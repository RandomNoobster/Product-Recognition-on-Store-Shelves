# Product-Recognition-on-Store-Shelves

This repository contains our solution for the project work in the course `Computer Vision and Image Processing M`.

The images used are located in the `/images` folder, and the code is located in the `step_a.py`, `step_b.py`, `step_c.py`, and `util.py` files.
Each step of the project can be performed independently by running their respective files.

The full task specification is located in `task.pdf`.
